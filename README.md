# my-shop

사용자가 상품을 검색하고, 좋아요를 저장하고, 장바구니/결제를 진행할 수 있는 커머스 웹 서비스입니다.

토스페이먼츠 위젯을 사용한 실제 결제 흐름까지 구현했습니다.
